    Задание 3.1.2:

    По созданным в пр.1 данным написать следующие запросы на фильтрацию:

    • Выведете все машины марки “Toyota” (или любой другой марки, которая у вас есть): BMW

    • Найти всех водителей с именем “Олег” (или любым другим именем на ваше усмотрение): Ivan

    • Взяв любого случайного владельца получить его id, и по этому id получить экземпляр удостоверения в виде объекта модели (можно в 2 запроса): id у 4го владельца

    • Вывести всех владельцев красных машин (или любого другого цвета, который у вас присутствует): black

    • Найти всех владельцев, чей год владения машиной начинается с 2010 (или любой другой год, который присутствует у вас в базе) c 2013
